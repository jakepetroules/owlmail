Building (for desktop platforms)

Requirements

  • A compiler:
    • GCC 4.5 (for Unix) OR
    • MinGW 4.5 with MSYS (for Windows) OR
    • Microsoft Visual C++ 2008 (for Windows)
    • It may be possible to use other compilers but at the moment this is not supported. GCC/MinGW are free and open source.
  • 7-Zip 9.20
  • Qt 4.7
  • Qt Creator 2.1
  • OpenSSL 1.0.0 binaries for Windows from Shining Light Productions

Always download the latest KSC Email Tracker source code from http://www.petroules.com/products/kscemailtracker. Versions downloaded from our Git repository may be untested at the time.

Instructions

Setting up 3rd party components

Setting up the 3rd party components (such as Qt Solutions) is relatively easy to do as scripts are provided. Navigate to the 3rdparty directory and run configure.bat for Windows or configure.sh for Unix.

Compiling KSC Email Tracker

The next step is to compile KSC Email Tracker itself. Navigate to the kscemailtracker directory and open the kscemailtracker.pro file in Qt Creator. Set up a shadow build for MinGW, MSVC or GCC when prompted, select either the debug or release configurations, and then click build.

Building distribution packages

To build the installers, run deploy/win32/deploy.bat for Windows and deploy/macx/deploy.sh for Mac OS X. Installers for Linux are not currently available.
